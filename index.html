Introduction

This page provides documentation for software and systems used in the neutron electric dipole moment (nEDM) experiment located at the Technische Universität München (FRM-II).

Available pages

  • System Overview - Set of pages describing the general functionality of many of the (non-software) systems used in the nEDM experiment.
  • nEDM Interface - Page describing the web interface used for control of devices and visualization of data in the experiment.
  • pynedm - Provides a set of python tools for controlling devices and communicating with the central CouchDB server used in the experiment.
